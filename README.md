# Mie-Gruneisen_EOS_WDM

"Applicability of the Mie-Grüneisen Equation of State Model to Warm Dense Matter Conditions", Felipe González-Cataldo and Burkhard Militzer. Contributions to Plasma Physics (2025).

In this work, we test the validity of the Mie-Gruneisen model at Warm Dense Matter Conditions, exploring how the Grüneisen parameter changes with volume and temperature for different materials.
